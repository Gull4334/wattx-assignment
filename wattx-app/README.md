# Getting Started with This Project

Kindly Follow the steps to run the project smoothly.

1) run => "npm install" to install the dependencies.

2) run => "node server.js" to run the server for api.
Server will start on port 4200

3) run => "npm start" to start the project.
app will start at port 3000

4) run => "npx cypress open" to run the test
